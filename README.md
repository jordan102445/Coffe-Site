# Дневна табла за кафе

React/Vite веб апликација за дневно договарање кој е станат, кој е за кафе, кој кафич го сака и кога стига.

## Start

1. Инсталирај Node.js.
2. Во овој folder изврши:

```powershell
npm install
npm run dev
```

Admin PIN за генерирање е `1200`. Копчето за генерирање е активно само за админ, за денешниот работен ден, од 12:00 натаму. Табовите го прикажуваат денешниот работен ден и следните работни денови; за викенд почнуваат од понеделник.
